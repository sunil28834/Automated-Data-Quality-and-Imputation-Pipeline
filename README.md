# Automated Data Quality & Imputation Pipeline

An industry-style data engineering project that automatically detects data quality issues and performs intelligent imputation using machine learning techniques.

## 🚀 Project Overview
Poor data quality can severely impact machine learning models and analytics. This system automatically analyzes datasets, detects quality issues, and fixes missing values using automated pipelines.

## 🎯 Features
- Data validation and profiling
- Missing value detection
- Outlier detection
- Machine learning based data imputation
- Data quality scoring
- FastAPI backend for automation
- Streamlit dashboard for visualization
- Airflow DAG for pipeline scheduling

## 🏗 Architecture

Dataset → Data Quality Pipeline → ML Imputation → API → Dashboard

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- FastAPI
- Streamlit
- Apache Airflow

## 📊 Dashboard Features
- Dataset preview
- Missing value analysis
- Statistical summary
- Quality score generation
- Pipeline execution
